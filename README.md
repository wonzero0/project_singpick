# 🎤 Sing Pick! 
### AI 기반 퍼스널 노래 추천 노래방 부스

> **"당신에게 가장 잘 어울리는 한 곡을 찾아드립니다."**
> 사용자의 음색과 음역대를 분석하여 최적의 노래를 추천하고 실시간 피드백을 제공하는 스마트 노래방 시스템입니다.

---

## 📝 프로젝트 개요
* **프로젝트명:** Sing Pick!
* **목표:** AI 기술을 활용한 개인 맞춤형 노래 추천 및 가창 분석
* **기간:** `2025. 09. 01` ~ `2026. 06. 12`
* **대상:** 선곡이 어려운 분, 내 목소리에 딱 맞는 곡을 찾고 싶은 분

---

## 🔥 주요 기능 (Key Features)

* **🎙️ 음역대 및 음색 분석:** 사용자의 목소리를 분석하여 고유의 보컬 데이터 파악
* **🤖 개인 맞춤 추천:** 분석된 데이터를 바탕으로 가장 부르기 적합한 곡 매칭
* **📊 실시간 가창 피드백:** 음정, 박자 데이터를 분석하여 시각적인 스코어 제공
* **📱 QR 모바일 결과지:** 분석된 리포트를 장소에 상관없이 QR로 간편 확인

---

## 🛠 기술 스택 (Tech Stack)

| 분류 | 기술 | 상세 설명 |
| :--- | :--- | :--- |
| **Backend** | `FastAPI`, `Uvicorn` | 고성능 비동기 API 서버 구축 및 Swagger UI 제공 |
| **AI / ML** | `TensorFlow`, `PyTorch` | 음성 분석 엔진 및 가창 데이터 학습 모델링 |
| **Frontend** | `HTML5`, `Swagger UI` | 인터랙티브 UI 구현 및 실시간 분석 대시보드 |
| **Database** | `MySQL` | 사용자 데이터, 가창 기록 및 추천 리스트 관리 |
| **Security** | `AES-256`, `Bcrypt`, `JWT` | 데이터 대칭키 암호화(AES), 비밀번호 해싱 및 인증 보안 |
| **Audio** | `yt-dlp`, `Librosa` | 유튜브 MR 추출 및 오디오 피처링 분석 |
